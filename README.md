# MEB-Music
Music-App von Kevin, Stefan und Vasilije.

MEB-Music ist eine App zur Verwaltung von eigenen Musik-Playlists basierend auf Youtube-Videos. Diese App basiert auf den Vorbildern "Spotify" und "Soundcloud".
